# 3D Print Files

This folder should contain the STL files for the custom enclosure.

## Files to add

- `case-bottom.stl` — Bottom half of the enclosure
- `case-top.stl` — Top half / lid
- `button-cap.stl` — Optional button cap for the tactile switch

## Recommended Print Settings

- **Filament:** PLA or PETG
- **Layer height:** 0.2 mm
- **Infill:** 20%
- **Supports:** Yes (for screw bosses and overhangs)
- **Orientation:** Print with flat surfaces on the bed
